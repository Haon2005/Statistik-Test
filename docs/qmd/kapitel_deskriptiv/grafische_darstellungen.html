1.3 Grafische Darstellungen

Deskriptive Statistik

Lernziele: Am Ende dieses Kapitels kannst du

  • Balkendiagramme, Kreisdiagramme, Histogramme und empirische Verteilungsfunktionen in Python erstellen.
  • entscheiden, welche Grafik für welchen Merkmalstyp und welche Fragestellung geeignet ist.
  • Grafiken korrekt beschriften und interpretieren.

Einführung

Eine Tabelle mit Häufigkeiten ist präzise – aber ein Bild sagt mehr als tausend Zahlen. Grafische Darstellungen helfen, Muster, Unterschiede und Ausreißer auf einen Blick zu erkennen.

Welche Grafik wählt man? Das hängt vom Merkmalstyp ab:

Grafik Geeignet für
Balkendiagramm nominale, ordinale, diskrete Merkmale (wenige Ausprägungen)
Kreisdiagramm nominale, ordinale Merkmale (wenige Ausprägungen, Anteile zeigen)
Histogramm kardinale Merkmale (viele Ausprägungen, klassiert)
Emp. Verteilungsfunktion ordinale, kardinale Merkmale
Boxplot kardinale Merkmale (Vergleiche, Ausreißer)

Balkendiagramm

Das Balkendiagramm stellt für jede Merkmalsausprägung einen Balken dar, dessen Höhe der absoluten oder relativen Häufigkeit entspricht. Die Balkenbreite hat keine Bedeutung.

Tipp

Wann? Immer wenn du nominale, ordinale oder diskrete Merkmale mit wenigen Ausprägungen visualisieren möchtest.

Python-Tipp: plt.bar(x, heights) zeichnet ein Balkendiagramm. value_counts() liefert direkt die Balkenhöhen. Mit .plot(kind='bar') geht es noch kürzer.

Balkendiagramme können auch gruppierten werden, um Vergleiche zwischen Gruppen zu zeigen:

Kreisdiagramm

Das Kreisdiagramm (Tortendiagramm) teilt einen Kreis in Sektoren auf – die Fläche jedes Sektors entspricht der relativen Häufigkeit.

Vorsicht

Kritik: Kreisdiagramme sind schwieriger zu lesen als Balkendiagramme, weil das Auge Flächen und Winkel schlechter vergleicht als Höhen. Sie eignen sich am besten, wenn man Anteile am Ganzen hervorheben will (z.B. „Gentoo macht 36 % aus”).

Histogramm

Das Histogramm zeigt die Häufigkeitsverteilung eines kardinalen Merkmals. Der Wertebereich wird in Klassen eingeteilt; jede Klasse bekommt einen Balken, dessen Fläche der relativen Häufigkeit entspricht.

Wichtig: Die Balkenhöhe ist die Häufigkeitsdichte f(K_j) = h(K_j) / \text{Breite}, nicht die relative Häufigkeit. Bei gleichen Klassenbreiten sind beide proportional – bei ungleichen Breiten muss man die Dichte verwenden.

Tipp

Interpretation: Die Adélie- und Chinstrap-Pinguine haben ein ähnliches Gewicht (Schwerpunkt um 3500–4000 g). Gentoo-Pinguine sind deutlich schwerer (Schwerpunkt um 5000–5500 g).

Python-Tipp: plt.hist(daten, bins=grenzen, density=True) erzeugt ein Histogramm mit Häufigkeitsdichte auf der y-Achse. Mit density=False werden absolute Häufigkeiten gezeigt.

Empirische Verteilungsfunktion

Die empirische Verteilungsfunktion (ECDF – Empirical Cumulative Distribution Function) gibt für jeden Wert x an, welcher Anteil der Beobachtungen höchstens so groß wie x ist.

Definition 1 (Empirische Verteilungsfunktion) F_n(x) = \frac{\text{Anzahl der Beobachtungen} \le x}{n} = \sum_{a_j \le x} h(a_j)

F_n ist eine monoton steigende, rechtsseitig stetige Treppenfunktion. Sie springt an jeder beobachteten Merkmalsausprägung um deren relative Häufigkeit.

Vorteil gegenüber Histogramm: Keine Klassenwahl nötig. Nachteil: Schwieriger zu lesen für Einsteiger.

Beispiel 1 (ECDF lesen) Am obigen Diagramm kann man direkt ablesen:

  • F_n(3500) \approx 0.39 → ca. 39 % der Adélie-Pinguine wiegen höchstens 3500 g.
  • Der Anteil zwischen 3000 g und 3500 g: F_n(3500) - F_n(3000) \approx 0.39 - 0.07 = 0.32, also ca. 32 %.

Ausblick: Boxplot

Der Boxplot ist eine kompakte Darstellung, die auf fünf Kennzahlen basiert: Minimum, 25%-Quartil, Median, 75%-Quartil und Maximum. Er eignet sich besonders gut, um mehrere Gruppen zu vergleichen. Wir behandeln ihn ausführlich in Kapitel 1.4 (Kennzahlen).

Zusammenfassung

  • Balkendiagramm: für nominale, ordinale und diskrete Merkmale; Balkenhöhe = (relative) Häufigkeit.
  • Kreisdiagramm: für Anteile am Ganzen; schlechter lesbar als Balkendiagramm.
  • Histogramm: für kardinale Merkmale; Balkenfläche = relative Häufigkeit; Balkenhöhe = Häufigkeitsdichte.
  • Empirische Verteilungsfunktion: F_n(x) = Anteil der Beobachtungen ≤ x; keine Klassenwahl nötig.
  • Boxplot: kompakt, gut für Gruppenvergleiche (→ Kapitel 1.4).

## Balkendiagramm

Das **Balkendiagramm** stellt für jede Merkmalsausprägung einen Balken dar, dessen Höhe der absoluten oder relativen Häufigkeit entspricht. Die Balkenbreite hat keine Bedeutung.

::: {.callout-tip}
**Wann?** Immer wenn du nominale, ordinale oder diskrete Merkmale mit wenigen Ausprägungen visualisieren möchtest.
:::

```{python}
#| echo: false
#| eval: true
#| fig-cap: "Balkendiagramm: Absolute Häufigkeiten der Pinguinarten"
#| label: fig-balken-arten
import matplotlib.pyplot as plt
import matplotlib
matplotlib.rcParams['figure.dpi'] = 120

try:
    from palmerpenguins import load_penguins
    penguins = load_penguins()
except:
    import pandas as pd
    import io, urllib.request
    url = "https://raw.githubusercontent.com/allisonhorst/palmerpenguins/main/inst/extdata/penguins.csv"
    penguins = pd.read_csv(url)

counts = penguins["species"].value_counts()

fig, axes = plt.subplots(1, 2, figsize=(10, 4))

# Absolute Häufigkeiten
axes[0].bar(counts.index, counts.values, color=["#4c72b0","#dd8452","#55a868"], edgecolor="white")
axes[0].set_title("Absolute Häufigkeiten")
axes[0].set_xlabel("Pinguinart")
axes[0].set_ylabel("Anzahl")
for i, v in enumerate(counts.values):
    axes[0].text(i, v + 1, str(v), ha='center', fontsize=10)

# Relative Häufigkeiten
rel = counts / counts.sum()
axes[1].bar(counts.index, rel.values, color=["#4c72b0","#dd8452","#55a868"], edgecolor="white")
axes[1].set_title("Relative Häufigkeiten")
axes[1].set_xlabel("Pinguinart")
axes[1].set_ylabel("Anteil")
axes[1].set_ylim(0, 0.55)
for i, v in enumerate(rel.values):
    axes[1].text(i, v + 0.005, f"{v:.2f}", ha='center', fontsize=10)

plt.tight_layout()
plt.savefig("balken_arten.png", bbox_inches='tight')
plt.show()
```

::: {.callout-python}
**Python-Tipp:** `plt.bar(x, heights)` zeichnet ein Balkendiagramm. `value_counts()` liefert direkt die Balkenhöhen. Mit `.plot(kind='bar')` geht es noch kürzer.
:::

Balkendiagramme können auch **gruppierten** werden, um Vergleiche zwischen Gruppen zu zeigen:

```{python}
#| echo: false
#| eval: true
#| fig-cap: "Gruppiertes Balkendiagramm: Pinguinarten je Insel"
#| label: fig-balken-grupiert
import matplotlib.pyplot as plt
import numpy as np

try:
    from palmerpenguins import load_penguins
    penguins = load_penguins()
except:
    import pandas as pd
    import urllib.request
    url = "https://raw.githubusercontent.com/allisonhorst/palmerpenguins/main/inst/extdata/penguins.csv"
    penguins = pd.read_csv(url)

import pandas as pd
kreuztab = pd.crosstab(penguins["island"], penguins["species"])

x = np.arange(len(kreuztab.index))
width = 0.25
colors = ["#4c72b0","#dd8452","#55a868"]

fig, ax = plt.subplots(figsize=(8, 4))
for i, (art, color) in enumerate(zip(kreuztab.columns, colors)):
    ax.bar(x + i*width, kreuztab[art], width, label=art, color=color)

ax.set_xticks(x + width)
ax.set_xticklabels(kreuztab.index)
ax.set_xlabel("Insel")
ax.set_ylabel("Anzahl")
ax.set_title("Pinguinarten je Insel")
ax.legend(title="Art")
plt.tight_layout()
plt.show()
```

## Kreisdiagramm

Das **Kreisdiagramm** (Tortendiagramm) teilt einen Kreis in Sektoren auf – die Fläche jedes Sektors entspricht der relativen Häufigkeit.

::: {.callout-caution}
**Kritik:** Kreisdiagramme sind schwieriger zu lesen als Balkendiagramme, weil das Auge Flächen und Winkel schlechter vergleicht als Höhen. Sie eignen sich am besten, wenn man **Anteile am Ganzen** hervorheben will (z.B. „Gentoo macht 36 % aus").
:::

```{python}
#| echo: false
#| eval: true
#| fig-cap: "Kreisdiagramm: Anteile der Pinguinarten"
#| label: fig-pie
import matplotlib.pyplot as plt

try:
    from palmerpenguins import load_penguins
    penguins = load_penguins()
except:
    import pandas as pd
    url = "https://raw.githubusercontent.com/allisonhorst/palmerpenguins/main/inst/extdata/penguins.csv"
    penguins = pd.read_csv(url)

counts = penguins["species"].value_counts()
fig, axes = plt.subplots(1, 2, figsize=(10, 4))

# Kreisdiagramm
axes[0].pie(counts.values, labels=counts.index,
            autopct='%1.1f%%', colors=["#4c72b0","#dd8452","#55a868"],
            startangle=90)
axes[0].set_title("Kreisdiagramm")

# Vergleich: Balkendiagramm
axes[1].bar(counts.index, counts.values / counts.sum(),
            color=["#4c72b0","#dd8452","#55a868"])
axes[1].set_title("Balkendiagramm (zum Vergleich)")
axes[1].set_ylabel("Relativer Anteil")
axes[1].set_ylim(0, 0.55)

plt.suptitle("Welche Darstellung ist übersichtlicher?", y=1.02)
plt.tight_layout()
plt.show()
```

## Histogramm

Das **Histogramm** zeigt die Häufigkeitsverteilung eines kardinalen Merkmals. Der Wertebereich wird in Klassen eingeteilt; jede Klasse bekommt einen Balken, dessen **Fläche** der relativen Häufigkeit entspricht.

::: {.callout-rechnen}
**Wichtig:** Die Balkenhöhe ist die **Häufigkeitsdichte** $f(K_j) = h(K_j) / \text{Breite}$, nicht die relative Häufigkeit. Bei gleichen Klassenbreiten sind beide proportional – bei ungleichen Breiten **muss** man die Dichte verwenden.
:::

```{python}
#| echo: false
#| eval: true
#| fig-cap: "Histogramme: Körpergewicht der drei Pinguinarten"
#| label: fig-histogramme
import matplotlib.pyplot as plt
import numpy as np

try:
    from palmerpenguins import load_penguins
    penguins = load_penguins()
except:
    import pandas as pd
    url = "https://raw.githubusercontent.com/allisonhorst/palmerpenguins/main/inst/extdata/penguins.csv"
    penguins = pd.read_csv(url)

arten = ["Adelie", "Chinstrap", "Gentoo"]
farben = ["#4c72b0", "#dd8452", "#55a868"]
grenzen = np.arange(2500, 6600, 500)

fig, axes = plt.subplots(1, 3, figsize=(12, 4), sharey=True)

for ax, art, farbe in zip(axes, arten, farben):
    daten = penguins[penguins["species"] == art]["body_mass_g"].dropna()
    ax.hist(daten, bins=grenzen, density=True, color=farbe, edgecolor="white", alpha=0.85)
    ax.set_title(art)
    ax.set_xlabel("Körpergewicht (g)")
    ax.set_xlim(2000, 7000)

axes[0].set_ylabel("Häufigkeitsdichte")
plt.suptitle("Körpergewicht je Pinguinart", y=1.02)
plt.tight_layout()
plt.show()
```

::: {.callout-tip}
**Interpretation:** Die Adélie- und Chinstrap-Pinguine haben ein ähnliches Gewicht (Schwerpunkt um 3500–4000 g). Gentoo-Pinguine sind deutlich schwerer (Schwerpunkt um 5000–5500 g).
:::

::: {.callout-python}

## Empirische Verteilungsfunktion

Die **empirische Verteilungsfunktion** (ECDF – *Empirical Cumulative Distribution Function*) gibt für jeden Wert $x$ an, welcher **Anteil** der Beobachtungen **höchstens so groß** wie $x$ ist.

::: {#def-ecdf}
### Empirische Verteilungsfunktion

$$F_n(x) = \frac{\text{Anzahl der Beobachtungen} \le x}{n} = \sum_{a_j \le x} h(a_j)$$

$F_n$ ist eine monoton steigende, rechtsseitig stetige Treppenfunktion. Sie springt an jeder beobachteten Merkmalsausprägung um deren relative Häufigkeit.
:::

**Vorteil gegenüber Histogramm:** Keine Klassenwahl nötig. **Nachteil:** Schwieriger zu lesen für Einsteiger.

```{python}
#| echo: false
#| eval: true
#| fig-cap: "Histogramm und ECDF: Körpergewicht der Adélie-Pinguine"
#| label: fig-ecdf
import matplotlib.pyplot as plt
import numpy as np

try:
    from palmerpenguins import load_penguins
    penguins = load_penguins()
except:
    import pandas as pd
    url = "https://raw.githubusercontent.com/allisonhorst/palmerpenguins/main/inst/extdata/penguins.csv"
    penguins = pd.read_csv(url)

adelie = penguins[penguins["species"] == "Adelie"]["body_mass_g"].dropna().sort_values()
n = len(adelie)

fig, axes = plt.subplots(1, 2, figsize=(10, 4))

# Histogramm
axes[0].hist(adelie, bins=np.arange(2500, 5500, 250), density=False,
             color="#4c72b0", edgecolor="white")
axes[0].set_xlabel("Körpergewicht (g)")
axes[0].set_ylabel("Absolute Häufigkeit")
axes[0].set_title("Histogramm")
axes[0].set_xlim(2200, 5500)

# ECDF
ecdf_x = np.concatenate([[adelie.iloc[0] - 1], adelie.values, [adelie.iloc[-1] + 1]])
ecdf_y = np.concatenate([[0], np.arange(1, n + 1) / n, [1]])
axes[1].step(ecdf_x, ecdf_y, where='post', color="#4c72b0", linewidth=2)
axes[1].scatter(adelie.values, np.arange(1, n + 1) / n, s=8, color="#4c72b0")
axes[1].set_xlabel("Körpergewicht (g)")
axes[1].set_ylabel("$F_n(x)$")
axes[1].set_title("Empirische Verteilungsfunktion")
axes[1].set_xlim(2200, 5500)
axes[1].axhline(0.5, color="gray", linestyle="--", alpha=0.5, label="50%")
axes[1].legend()

plt.suptitle("Adélie-Pinguine", y=1.02)
plt.tight_layout()
plt.show()
```

::: {#exm-ecdf-lesen}
### ECDF lesen

Am obigen Diagramm kann man direkt ablesen:

- $F_n(3500) \approx 0.39$ → ca. 39 % der Adélie-Pinguine wiegen **höchstens** 3500 g.
- Der Anteil zwischen 3000 g und 3500 g: $F_n(3500) - F_n(3000) \approx 0.39 - 0.07 = 0.32$, also ca. 32 %.
:::

```{pyodide-python}
#| packages: ["palmerpenguins"]
import matplotlib.pyplot as plt
import numpy as np
from palmerpenguins import load_penguins

penguins = load_penguins()

# Erstelle die ECDF für die Flossenlänge aller Pinguine
daten = penguins["flipper_length_mm"].dropna().sort_values()
n = len(daten)

ecdf_y = np.arange(1, n + 1) / n

plt.figure(figsize=(7, 4))
plt.step(daten, ecdf_y, where='post', color="steelblue", linewidth=2)
plt.xlabel("Flossenlänge (mm)")
plt.ylabel("F_n(x)")
plt.title("Empirische Verteilungsfunktion: Flossenlänge")
plt.grid(True, alpha=0.3)
plt.tight_layout()
plt.show()

# Wie viele Pinguine haben eine Flossenlänge von höchstens 200 mm?
anteil = (daten <= 200).mean()
print(f"Anteil Pinguine mit Flossenlänge ≤ 200 mm: {anteil:.3f} ({anteil*100:.1f}%)")
```

## Ausblick: Boxplot

Der **Boxplot** ist eine kompakte Darstellung, die auf fünf Kennzahlen basiert: Minimum, 25%-Quartil, Median, 75%-Quartil und Maximum. Er eignet sich besonders gut, um mehrere Gruppen zu vergleichen. Wir behandeln ihn ausführlich in Kapitel 1.4 (Kennzahlen).

```{python}
#| echo: false
#| eval: true
#| fig-cap: "Boxplots: Schnabellänge je Pinguinart"
#| label: fig-boxplot-vorschau
import matplotlib.pyplot as plt

try:
    from palmerpenguins import load_penguins
    penguins = load_penguins()
except:
    import pandas as pd
    url = "https://raw.githubusercontent.com/allisonhorst/palmerpenguins/main/inst/extdata/penguins.csv"
    penguins = pd.read_csv(url)

arten = ["Adelie", "Chinstrap", "Gentoo"]
daten = [penguins[penguins["species"] == art]["bill_length_mm"].dropna().values
         for art in arten]

fig, ax = plt.subplots(figsize=(7, 4))
bp = ax.boxplot(daten, labels=arten, patch_artist=True,
                boxprops=dict(facecolor="#d0e4f5"), medianprops=dict(color="#2c5f8a", linewidth=2))
ax.set_xlabel("Pinguinart")
ax.set_ylabel("Schnabellänge (mm)")
ax.set_title("Boxplots: Schnabellänge je Art")
plt.tight_layout()
plt.show()
```

## Zusammenfassung

- **Balkendiagramm:** für nominale, ordinale und diskrete Merkmale; Balkenhöhe = (relative) Häufigkeit.
- **Kreisdiagramm:** für Anteile am Ganzen; schlechter lesbar als Balkendiagramm.
- **Histogramm:** für kardinale Merkmale; Balkenfläche = relative Häufigkeit; Balkenhöhe = Häufigkeitsdichte.
- **Empirische Verteilungsfunktion:** $F_n(x)$ = Anteil der Beobachtungen ≤ $x$; keine Klassenwahl nötig.
- **Boxplot:** kompakt, gut für Gruppenvergleiche (→ Kapitel 1.4).
