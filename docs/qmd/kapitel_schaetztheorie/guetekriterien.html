2.3 Gütekriterien für Punktschätzer

Lernziele

Am Ende dieses Unterkapitels kannst du:

  • die drei wichtigsten Gütekriterien für Schätzer (Erwartungstreue, Effizienz, Konsistenz) erläutern und visuell interpretieren.
  • den Bias eines Schätzers berechnen und erklären, warum s^2 mit 1/(n-1) erwartungstreu ist.
  • den Standardfehler des Stichprobenmittels \sigma/\sqrt{n} interpretieren und berechnen.
  • Konsistenz anhand von Simulationen erkennen.

Einführung: Was macht einen guten Schätzer aus?

Wie wir in Kapitel 2.1 gesehen haben, liefert ein Schätzer bei jeder neuen Stichprobe einen anderen Schätzwert. Wann ist ein Schätzer gut?

Ein anschauliches Bild: Stell dir einen Bogenschützen vor, der auf eine Zielscheibe schießt. Das Zentrum der Scheibe ist der wahre Parameter \theta. Jeder Pfeil ist ein Schätzwert aus einer zufälligen Stichprobe.

Wir wollen einen Schätzer, der: 1. Im Mittel den wahren Wert trifft (Erwartungstreue), 2. Wenig streut (Effizienz), 3. Für mehr Daten immer genauer wird (Konsistenz).

Erwartungstreue (Bias)

HinweisDefinition: Bias und Erwartungstreue

Der Bias (oder die Verzerrung) eines Schätzers \hat{\theta} für den Parameter \theta ist der systematische Schätzfehler: \text{Bias}(\hat{\theta}) = E(\hat{\theta}) - \theta

Ein Schätzer heißt erwartungstreu (oder unverzerrt), wenn sein Bias null ist: E(\hat{\theta}) = \theta

Das Stichprobenmittel ist erwartungstreu

Für unabhängige, identisch verteilte Werte mit Erwartungswert \mu gilt: E(\overline{X}_n) = E\left(\frac{1}{n}\sum_{i=1}^n X_i\right) = \frac{1}{n}\sum_{i=1}^n E(X_i) = \mu

Das Stichprobenmittel ist ein erwartungstreuer Schätzer für \mu.

Warum s^2 den Faktor 1/(n-1) braucht

Wenn man die Varianz mit 1/n berechnet: \tilde{s}^2 = \frac{1}{n}\sum_{i=1}^n (x_i - \bar{x})^2

dann gilt E(\tilde{s}^2) = \frac{n-1}{n} \sigma^2 < \sigma^2 – der Schätzer unterschätzt systematisch die wahre Varianz \sigma^2. Mit dem Faktor 1/(n-1) hingegen gilt: E(s^2) = \sigma^2 \quad \text{(erwartungstreu!)}

VorsichtHäufiger Fehler: numpy vs. pandas Varianz

In Python berechnen numpy und pandas die Varianz unterschiedlich:

import numpy as np, pandas as pd

x = [2, 4, 6, 8]
np.var(x)       # 1/n → NICHT erwartungstreu (= 5.0)
np.var(x, ddof=1)  # 1/(n-1) → erwartungstreu (= 6.667)
pd.Series(x).var()  # 1/(n-1) → erwartungstreu (= 6.667)

Für Schätzung der Populationsvarianz immer ddof=1 verwenden!

Simulation: Bias sichtbar machen

Bei kleinen Stichproben (n = 5) schätzt \tilde{s}^2 die Varianz deutlich zu klein. Der Unterschied wird mit wachsendem n kleiner.

Effizienz und Standardfehler

Ein Schätzer ist effizient, wenn er wenig streut. Das zentrale Maß dafür ist der Standardfehler.

HinweisDefinition: Standardfehler

Der Standardfehler eines Schätzers \hat{\theta} ist seine Standardabweichung: \text{SE}(\hat{\theta}) = \sqrt{\text{Var}(\hat{\theta})}

Er misst, wie stark die Schätzwerte zwischen verschiedenen Stichproben streuen.

Standardfehler des Stichprobenmittels

Für unabhängige, identisch verteilte Werte mit Varianz \sigma^2 gilt: \text{SE}(\overline{X}_n) = \frac{\sigma}{\sqrt{n}}

Herleitung: \text{Var}(\overline{X}_n) = \text{Var}\!\left(\frac{1}{n}\sum_{i=1}^n X_i\right) = \frac{1}{n^2}\sum_{i=1}^n \text{Var}(X_i) = \frac{n \cdot \sigma^2}{n^2} = \frac{\sigma^2}{n}

Fazit: Mit n-mal so vielen Daten halbiert sich nicht der Standardfehler – er fällt nur um den Faktor \sqrt{n}. Um die Genauigkeit zu verdoppeln, braucht man die vierfache Datenmenge!

Standardfehler in Python schätzen

Da \sigma in der Praxis unbekannt ist, schätzt man den Standardfehler durch: \widehat{\text{SE}}(\overline{X}_n) = \frac{s}{\sqrt{n}}

Konsistenz

HinweisDefinition: Konsistenz

Eine Folge von Schätzern (\hat{\theta}_n)_{n \in \mathbb{N}} heißt konsistent, wenn die Schätzwerte für wachsenden Stichprobenumfang n gegen den wahren Parameterwert konvergieren:

\hat{\theta}_n \xrightarrow{n \to \infty} \theta \quad \text{(stochastisch)}

Anschaulich: Bei genug Daten wird die Schätzung beliebig genau.

Das Stichprobenmittel ist konsistent, denn \text{SE}(\overline{X}_n) = \sigma/\sqrt{n} \to 0 für n \to \infty. Das ist eine Folgerung aus dem Gesetz der großen Zahlen.

Simulation: Konvergenz des Stichprobenmittels

Das Stichprobenmittel pendelt zunächst stark, nähert sich dann aber immer enger dem wahren Wert an. Das ±2·SE-Band zeigt, wie sich die Unsicherheit mit 1/\sqrt{n} verringert.

Der MSE: Bias und Varianz zusammen

Der mittlere quadratische Fehler (MSE) vereint Bias und Varianz in einer Größe:

\text{MSE}(\hat{\theta}) = \text{Var}(\hat{\theta}) + \text{Bias}(\hat{\theta})^2

Das nennt sich die Bias-Varianz-Zerlegung. Ein guter Schätzer hat sowohl kleinen Bias als auch kleine Varianz – beide Terme tragen zum Gesamtfehler bei.

Zusammenfassung

Kriterium Was wird bewertet? Formel / Bedingung
Erwartungstreue Systematischer Fehler E(\hat{\theta}) = \theta (Bias = 0)
Standardfehler Streuung der Schätzwerte \text{SE} = \sqrt{\text{Var}(\hat{\theta})}
Konsistenz Verhalten für n \to \infty \hat{\theta}_n \to \theta
MSE Gesamtfehler \text{MSE} = \text{Var} + \text{Bias}^2

Für die Praxis gilt: Die Standardschätzer \overline{X}_n (für \mu) und s^2 (für \sigma^2) sind erwartungstreu und konsistent. Der Standardfehler des Mittelwerts beträgt \sigma/\sqrt{n}.

```{python}
#| echo: false
#| eval: true
np.random.seed(42)

def zielscheibe(ax, mu, sigma, title, n=80):
    pts = np.random.normal(mu, sigma, (n, 2))
    # Kreise
    for r, c in [(1,'#e9ecef'),(2,'#dee2e6'),(3,'#ced4da'),(4,'#adb5bd')]:
        circle = plt.Circle((0, 0), r, color=c, fill=True)
        ax.add_patch(circle)
    ax.set_xlim(-5, 5); ax.set_ylim(-5, 5)
    ax.set_aspect('equal')
    ax.plot(*([0],[0]), 'k+', ms=12, mew=2.5, zorder=5)
    ax.scatter(pts[:,0], pts[:,1], s=20, color='#ae2012', alpha=0.7, zorder=4)
    # Schwerpunkt
    ax.plot(pts[:,0].mean(), pts[:,1].mean(), 'o', ms=10, color='#0a9396',
            markeredgecolor='white', markeredgewidth=1.5, zorder=6,
            label=f'Mittelpunkt der Treffer')
    ax.set_xticks([]); ax.set_yticks([])
    ax.set_title(title, fontsize=10)

fig, axes = plt.subplots(1, 4, figsize=(12, 3.5))

zielscheibe(axes[0], mu=[0,0], sigma=0.8, title='Erwartungstreu\n& effizient')
zielscheibe(axes[1], mu=[0,0], sigma=2.0, title='Erwartungstreu\n& ineffizient')
zielscheibe(axes[2], mu=[2,1], sigma=0.8, title='Nicht erw.treu\n& effizient (Bias!)')
zielscheibe(axes[3], mu=[2,1], sigma=2.0, title='Nicht erw.treu\n& ineffizient')

for ax in axes:
    ax.legend(loc='lower right', fontsize=7, framealpha=0.8)

plt.suptitle('Vier Schätzer im Vergleich: Zielscheiben-Metapher', y=1.02, fontsize=11)
plt.tight_layout()
plt.show()
```

### Simulation: Bias sichtbar machen

```{python}
#| echo: false
#| eval: true
np.random.seed(1)
sigma2_true = 4.0
N_sims = 500
ns = [5, 10, 20, 50, 100]

biased_means = []
unbiased_means = []

for n in ns:
    biased = [np.var(np.random.normal(0, 2, n)) for _ in range(N_sims)]
    unbiased = [np.var(np.random.normal(0, 2, n), ddof=1) for _ in range(N_sims)]
    biased_means.append(np.mean(biased))
    unbiased_means.append(np.mean(unbiased))

fig, ax = plt.subplots(figsize=(7, 4))
ax.plot(ns, biased_means, 'o-', color='#ae2012', lw=2, label='$\\tilde{s}^2$ (mit $1/n$) – verzerrt')
ax.plot(ns, unbiased_means, 's-', color='#0a9396', lw=2, label='$s^2$ (mit $1/(n-1)$) – erwartungstreu')
ax.axhline(sigma2_true, color='k', ls='--', lw=1.5, label=f'Wahrer Wert $\\sigma^2 = {sigma2_true}$')
ax.set_xlabel('Stichprobenumfang $n$')
ax.set_ylabel('Mittlerer Schätzwert (aus 500 Stichproben)')
ax.set_title('Bias-Vergleich der beiden Varianz-Schätzer')
ax.legend(fontsize=9)
plt.tight_layout()
plt.show()
```

Bei kleinen Stichproben ($n = 5$) schätzt $\tilde{s}^2$ die Varianz deutlich zu klein.
Der Unterschied wird mit wachsendem $n$ kleiner.

## Effizienz und Standardfehler

Ein Schätzer ist **effizient**, wenn er wenig streut. Das zentrale Maß dafür ist der
**Standardfehler**.

::: {.callout-note icon=false}
## Definition: Standardfehler

Der **Standardfehler** eines Schätzers $\hat{\theta}$ ist seine Standardabweichung:
$$\text{SE}(\hat{\theta}) = \sqrt{\text{Var}(\hat{\theta})}$$

Er misst, wie stark die Schätzwerte zwischen verschiedenen Stichproben streuen.
:::

### Standardfehler des Stichprobenmittels

Für unabhängige, identisch verteilte Werte mit Varianz $\sigma^2$ gilt:
$$\text{SE}(\overline{X}_n) = \frac{\sigma}{\sqrt{n}}$$

::: {.callout-rechnen appearance="minimal"}
**Herleitung:**
$$\text{Var}(\overline{X}_n) = \text{Var}\!\left(\frac{1}{n}\sum_{i=1}^n X_i\right)
= \frac{1}{n^2}\sum_{i=1}^n \text{Var}(X_i) = \frac{n \cdot \sigma^2}{n^2} = \frac{\sigma^2}{n}$$
:::

```{python}
#| echo: false
#| eval: true
np.random.seed(3)
true_mu = 3700
true_sigma = 460

fig, axes = plt.subplots(1, 3, figsize=(11, 3.8))
ns_plot = [10, 30, 100]
colors = ['#ca6702', '#0a9396', '#005f73']

for ax, n, col in zip(axes, ns_plot, colors):
    estimates = [np.mean(np.random.normal(true_mu, true_sigma, n)) for _ in range(500)]
    ax.hist(estimates, bins=25, color=col, edgecolor='white', alpha=0.8, density=True)
    se = true_sigma / np.sqrt(n)
    xr = np.linspace(true_mu - 4*se, true_mu + 4*se, 300)
    ax.plot(xr, stats.norm.pdf(xr, true_mu, se), 'k', lw=2)
    ax.axvline(true_mu, color='#ae2012', lw=2, ls='--')
    ax.set_title(f'n = {n}\nSE = σ/√n = {se:.1f} g', fontsize=10)
    ax.set_xlabel('x̄ (g)')
    ax.set_xlim(2800, 4600)
    if ax == axes[0]:
        ax.set_ylabel('Dichte')

plt.suptitle('Verteilung von $\\bar{X}_n$ für verschiedene Stichprobengrößen', y=1.02, fontsize=11)
plt.tight_layout()
plt.show()
```

**Fazit:** Mit $n$-mal so vielen Daten halbiert sich nicht der Standardfehler – er fällt nur um
den Faktor $\sqrt{n}$. Um die Genauigkeit zu verdoppeln, braucht man die **vierfache** Datenmenge!

### Standardfehler in Python schätzen

Da $\sigma$ in der Praxis unbekannt ist, schätzt man den Standardfehler durch:
$$\widehat{\text{SE}}(\overline{X}_n) = \frac{s}{\sqrt{n}}$$

```{pyodide-python}
import pandas as pd
import numpy as np
from pyodide.http import open_url

penguins = pd.read_csv(open_url("https://raw.githubusercontent.com/allisonhorst/palmerpenguins/main/inst/extdata/penguins.csv"))
masse = penguins[penguins['species'] == 'Adelie']['body_mass_g'].dropna()

n = len(masse)
s = masse.std()           # Stichproben-Standardabweichung (mit n-1)
se = s / np.sqrt(n)       # geschätzter Standardfehler

print(f"n = {n}")
print(f"s = {s:.1f} g")
print(f"Geschätzter Standardfehler SE = s/√n = {se:.1f} g")
```

## Konsistenz

::: {.callout-note icon=false}
## Definition: Konsistenz

Eine Folge von Schätzern $(\hat{\theta}_n)_{n \in \mathbb{N}}$ heißt **konsistent**, wenn die
Schätzwerte für wachsenden Stichprobenumfang $n$ gegen den wahren Parameterwert konvergieren:

$$\hat{\theta}_n \xrightarrow{n \to \infty} \theta \quad \text{(stochastisch)}$$

Anschaulich: Bei genug Daten wird die Schätzung beliebig genau.
:::

Das Stichprobenmittel ist konsistent, denn $\text{SE}(\overline{X}_n) = \sigma/\sqrt{n} \to 0$
für $n \to \infty$. Das ist eine Folgerung aus dem **Gesetz der großen Zahlen**.

### Simulation: Konvergenz des Stichprobenmittels

```{python}
#| echo: false
#| eval: true
np.random.seed(5)
true_mu = 3700.0
true_sigma = 460.0

max_n = 300
running_means = np.cumsum(np.random.normal(true_mu, true_sigma, max_n)) / np.arange(1, max_n + 1)

fig, ax = plt.subplots(figsize=(9, 4))
ns = np.arange(1, max_n + 1)
ax.plot(ns, running_means, '#0a9396', lw=1.8, label='laufender Mittelwert $\\bar{x}_n$')
ax.axhline(true_mu, color='#ae2012', lw=2, ls='--', label=f'Wahrer Wert μ = {true_mu:.0f} g')

se = true_sigma / np.sqrt(ns)
ax.fill_between(ns, true_mu - 2*se, true_mu + 2*se, alpha=0.15, color='#ae2012',
                label='±2·SE Band')

ax.set_xlabel('Stichprobenumfang n')
ax.set_ylabel('x̄ (g)')
ax.set_title('Konsistenz: $\\bar{X}_n$ konvergiert gegen μ für $n \\to \\infty$')
ax.legend(fontsize=9)
ax.set_xlim(1, max_n)
plt.tight_layout()
plt.show()
```

Das Stichprobenmittel pendelt zunächst stark, nähert sich dann aber immer enger dem wahren
Wert an. Das ±2·SE-Band zeigt, wie sich die Unsicherheit mit $1/\sqrt{n}$ verringert.

## Der MSE: Bias und Varianz zusammen

Der **mittlere quadratische Fehler** (MSE) vereint Bias und Varianz in einer Größe:

$$\text{MSE}(\hat{\theta}) = \text{Var}(\hat{\theta}) + \text{Bias}(\hat{\theta})^2$$

Das nennt sich die **Bias-Varianz-Zerlegung**. Ein guter Schätzer hat sowohl kleinen Bias als
auch kleine Varianz – beide Terme tragen zum Gesamtfehler bei.

```{python}
#| echo: false
#| eval: true
ns = np.arange(2, 101)

# Biased (1/n) vs. unbiased (1/(n-1)) Schätzer für σ² = 4
sigma2 = 4.0

# Bias²: E(~s²) = (n-1)/n * σ², also Bias = -σ²/n
bias_sq = (sigma2 / ns)**2
var_biased = 2 * sigma2**2 / ns**2  # Approx. für Var(~s²) bei Normalverteilung: 2σ⁴/(n)
var_unbiased = 2 * sigma2**2 / (ns - 1)  # 2σ⁴/(n-1)

mse_biased = bias_sq + var_biased
mse_unbiased = var_unbiased  # bias=0

fig, axes = plt.subplots(1, 2, figsize=(10, 3.8))

ax = axes[0]
ax.plot(ns, bias_sq, '#ae2012', lw=2, label='Bias²')
ax.plot(ns, var_biased, '#0a9396', lw=2, label='Varianz')
ax.plot(ns, mse_biased, 'k', lw=2, ls='--', label='MSE = Bias² + Varianz')
ax.set_title('Verzerrt: $\\tilde{s}^2$ (mit $1/n$)', fontsize=10)
ax.set_xlabel('n'); ax.set_ylabel('Fehler'); ax.legend(fontsize=8); ax.set_xlim(2, 100)

ax2 = axes[1]
ax2.plot(ns, np.zeros_like(ns), '#ae2012', lw=2, label='Bias² = 0')
ax2.plot(ns, var_unbiased, '#0a9396', lw=2, label='Varianz = MSE')
ax2.set_title('Erwartungstreu: $s^2$ (mit $1/(n-1)$)', fontsize=10)
ax2.set_xlabel('n'); ax2.legend(fontsize=8); ax2.set_xlim(2, 100)

plt.suptitle('MSE-Zerlegung für biased vs. unbiased Varianz-Schätzer', fontsize=11, y=1.02)
plt.tight_layout()
plt.show()
```

## Zusammenfassung

| Kriterium | Was wird bewertet? | Formel / Bedingung |
|-----------|--------------------|--------------------|
| **Erwartungstreue** | Systematischer Fehler | $E(\hat{\theta}) = \theta$ (Bias = 0) |
| **Standardfehler** | Streuung der Schätzwerte | $\text{SE} = \sqrt{\text{Var}(\hat{\theta})}$ |
| **Konsistenz** | Verhalten für $n \to \infty$ | $\hat{\theta}_n \to \theta$ |
| **MSE** | Gesamtfehler | $\text{MSE} = \text{Var} + \text{Bias}^2$ |

Für die Praxis gilt: Die Standardschätzer $\overline{X}_n$ (für $\mu$) und $s^2$ (für $\sigma^2$)
sind erwartungstreu und konsistent. Der Standardfehler des Mittelwerts beträgt $\sigma/\sqrt{n}$.
